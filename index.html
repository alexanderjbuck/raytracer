Raytracer

Seeded dungeon explorer

WASD / arrows to move. Click the view and use the mouse to look.

Source on GitHub

How to play

  • Enter a game code or roll a random one. The same code always generates the same dungeon.
  • Explore the 64×64 maze of rooms and corridors from a first-person view.
  • Find the key somewhere in the reachable maze.
  • Reach the exit on the outer wall. It stays locked until you have the key.
  • Move with WASD or arrow keys. Click the game view to capture the mouse and look around.
